Apresentação do Sistema

Sobre o Projeto

Este sistema foi desenvolvido com o objetivo de realizar o controle de estoque de computadores, permitindo o cadastro, edição, listagem e exclusão de produtos de forma simples e eficiente.

Funcionalidades

  • Cadastro de produtos
  • Listagem organizada com rolagem controlada
  • Edição e exclusão de registros
  • Controle automático de status em estoque

Tecnologias Utilizadas

  • Java (Servlets e JSP)
  • JSTL
  • MySQL
  • HTML5 e CSS3
